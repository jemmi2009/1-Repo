## 🚀 Quick Start

### Prerequisites
- **Node.js** (v18+) or **Python** (v3.10+)
- **Git** (v2.0+)
- **AI Backend** (pick one):
  - 🏠 [Ollama](https://ollama.ai) - Local & private
  - ☁️ OpenAI API - Cloud-powered
- **System**: 4GB RAM (8GB recommended) — macOS, Linux, or Windows WSL2

### Installation (< 2 minutes)

```bash
# Clone and setup
git clone https://github.com/jemmi2009/1-Repo.git
cd 1-Repo

# Install dependencies
npm install
# or
pip install -r requirements.txt

# Configure AI backend
cp .env.example .env
# Edit .env with your Ollama endpoint or OpenAI key
```

### Get Started

```bash
# Launch the assistant
npm start
# or
python main.py

# Pro tips:
# • Press Ctrl+Shift+A anytime to capture tasks
# • Open http://localhost:3000 for the beautiful dashboard
# • Check docs/ for advanced configurations
```
